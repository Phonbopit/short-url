# short-url
Simple URL Shortener with Node.js and Hapi.js


## Prerequisite

- MongoDB
- Babel CLI

Install with : 

```
brew install mongodb
npm install babel-cli -g
```

## Usage

```js
npm install

npm run build
# or development mode with:
npm run dev

npm start
```

Open [http://localhost:5555/](http://localhost:5555) and have fun :)
